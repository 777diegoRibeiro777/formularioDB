## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML
- CSS
- JavaScript
- Git e Github
- PHP
- XAMPP

## 💻 Projeto

O objetivo do projeto era desenvolver duas páginas (Ambas utilizando HTML, CSS e JS), sendo uma delas funcional (O formulário) e a outra não (A tela de login). Para o fomulário, foi o utilizado o XAMPP afim de simular um banco de dados ja na integração foi escolhido o PHP para a realização desta tarefa.
